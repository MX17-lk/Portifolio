Corrida Multiplayer

Neste projeto, desenvolvi uma simulação de corrida entre quatro personagens, cada um posicionado em uma pista diferente. O movimento acontece de forma automática e aleatória, o que torna cada corrida imprevisível e divertida de assistir. A linha de chegada é representada por uma faixa vermelha no final da tela, e os corredores vão avançando em velocidades diferentes a cada segundo. O projeto explora conceitos como aleatoriedade, movimentação contínua e controle de múltiplos elementos na tela, oferecendo uma experiência visual simples, porém dinâmica.

Quente e Frio

O jogo “Quente e Frio” é uma experiência interativa em que o jogador deve encontrar um ponto secreto escondido na tela. A única dica disponível é a cor de fundo, que muda conforme a proximidade do cursor com o ponto-alvo: tons frios indicam distância, enquanto tons quentes revelam que o jogador está se aproximando. Quando o ponto é encontrado, a tela muda completamente, indicando a vitória. Esse projeto trabalha com lógica de distância, interatividade com o mouse e percepção visual, criando um desafio simples, mas envolvente.

Corrida Multiplayer

Desenvolvido com a biblioteca p5.js, este projeto apresenta uma corrida divertida entre quatro personagens coloridos, cada um em sua própria pista. A movimentação é realizada de forma automática, com velocidades aleatórias a cada quadro, o que garante uma experiência dinâmica e imprevisível para quem assiste. A linha de chegada, marcada por uma faixa vermelha, delimita o objetivo da corrida e traz tensão à competição. O projeto se destaca pelo uso eficaz de variáveis independentes, atualização constante no draw() e aplicação da função random() para gerar variações naturais no movimento. A organização espacial e o uso criativo de formas geométricas simples mostram uma boa compreensão do plano cartesiano e da lógica visual. É um ótimo exemplo de como ideias simples, quando bem estruturadas, podem se transformar em experiências cativantes. A simulação é divertida, bem construída e mostra domínio sobre conceitos básicos de programação criativa.

Minha Biblioteca: Uma Webpage Personalizada

Também desenvolvido em p5.js, este jogo interativo demonstra com clareza sua habilidade em conectar lógica de programação com percepção visual. A proposta é simples e inteligente: o jogador precisa encontrar um ponto secreto escondido na tela, usando apenas a cor de fundo como dica. À medida que o mouse se aproxima do alvo, o fundo esquenta (com tons avermelhados); ao se afastar, esfria (tons azulados). A distância é calculada em tempo real com a função dist(), e a alteração das cores acontece por meio de lógica condicional baseada nessa medição. O projeto mostra domínio do uso de coordenadas, interação com o mouse (mouseX, mouseY), e sensibilidade estética na transição de cores. Além de funcional, o jogo é visualmente agradável, intuitivo e envolvente — uma aplicação excelente de lógica matemática com criatividade.

Este projeto é uma releitura da famosa pintura “Monalisa”, feita com formas básicas usando p5.js. A técnica empregada é a de pixel art: pequenos retângulos são posicionados cuidadosamente no plano cartesiano para compor uma versão digital e minimalista da obra. O destaque aqui vai para a paciência e precisão na construção visual, além da criatividade em transformar arte clássica em linguagem de código. O projeto demonstra sensibilidade artística, domínio da estrutura de coordenadas e capacidade de planejar visualmente uma composição complexa a partir de elementos simples. É um excelente exemplo de como programação também pode ser uma forma de expressão estética.
